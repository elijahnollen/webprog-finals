# Elijah Crisehea Nollen - Portfolio Website

This repository contains my personal portfolio and guestbook project for Web Programming Finals.

## Overview
The portfolio is designed as an interactive, visual-first experience that combines:
- Personal profile and background
- Education timeline
- IT experience and skills
- Goals and aspirations
- Hobbies showcase
- Picture gallery
- Live guestbook wall

## Tech Stack
- Frontend: React, TypeScript, CSS
- Backend: NestJS
- Database: Supabase
- Hosting: Vercel

## Project Structure
- `frontend/` - portfolio UI and interactions
- `backend/` - guestbook API services
- `introduction/` - initial static reference files
